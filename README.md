# Classic Way • Technical Industrial Procurement
UAE's premier destination for high-standard industrial electrical components and marine hardware.

## 🏗️ Technical Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Type Safety**: TypeScript

## 🚀 Quick Start
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🌐 Deployment
This project is pre-configured for **Vercel**.
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Configuration**: `vercel.json` handles SPA routing resets.

## 📑 Project Structure
- `/src/pages`: Main application views (Home, Catalogue, Contact).
- `/src/components`: Reusable UI components (Navbar, Footer, Modals).
- `/src/data`: Technical product definitions and categories.
- `/public`: High-fidelity industrial assets.
